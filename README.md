## 🖋️ 커밋 컨벤션

| **태그**  |           **설명**            |
| :-------: | :---------------------------: |
|   feat   |   새로운 기능을 구현한 경우   |
|    fix   |     버그 수정, 오류 해결      |
|  refactor |     코드 수정 및 리팩토링     |
|  style |      스타일 추가 및 수정      |
|    docs  |     문서 추가, 수정, 삭제     |
|    init   |      개발 환경 초기 세팅      |
|   chore  | 그 외 자잘한 수정 (주석 삭제) |
